# Go Basics and Data Manipulation

## Overview
This project contains utility functions in Go for working with strings, slices, pointers, sorting, and numerical computations.

## Tech stack
Go, data structures, basic algorithms, pointers, slices

## Goal
Practice core Go programming concepts through small self-contained tasks involving memory manipulation, slice operations, sorting, string processing, and mathematical utilities.

## What was done
- implemented indexed access and slicing operations for strings
- worked with rune-based string handling
- manipulated pointers and memory offsets with `unsafe`
- implemented comparison and root computation for complex numbers
- implemented in-place heap sort
- implemented slice reversal, equality checks, and element deletion
- practiced pointer swapping and basic utility design in Go

## Result
Built a compact collection of Go utilities covering core language features, low-level memory handling, and common data manipulation tasks.
