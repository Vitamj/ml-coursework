# Filesystem Command Shell

## Overview
This project implements a console-based file system manager designed with object-oriented principles, SOLID, and extensible command parsing.

## Tech stack
C#, object-oriented programming, SOLID, design patterns, command parsing

## Goal
Build a modular application for navigating and managing a file system through commands, while keeping business logic decoupled from console input/output and storage implementation.

## What was done
- implemented command-based interaction with the file system
- supported navigation through relative and absolute paths
- implemented file operations: show, move, copy, delete, and rename
- designed tree-based directory listing with configurable depth
- separated core logic from presentation logic
- structured the system for extensibility under SOLID principles
- kept file system and output logic decoupled from console-specific code


## Result
Built a modular console file system application with extensible architecture, separation of concerns, and support for core navigation and file management workflows.
