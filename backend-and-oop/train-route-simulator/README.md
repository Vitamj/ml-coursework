# Train Route Simulator

## Overview
This project implements an object-oriented simulator for train route traversal in a simplified automated transport system.


## Tech stack
C#, object-oriented programming, simulation, unit testing

## Goal
Model train movement through route segments, stations, and force-based track sections while validating route completion under speed and system constraints.

## What was done
- designed an object-oriented model for trains and route segments
- implemented train movement based on speed, acceleration, and applied force
- simulated traversal of regular tracks, force tracks, and stations
- modeled successful and failed route outcomes
- validated route completion under speed limits and stopping constraints
- preserved immutability of route-related entities
- covered business scenarios with unit tests

## Result
Built a modular transport route simulator that models train dynamics, route traversal, and failure conditions in an object-oriented way.
