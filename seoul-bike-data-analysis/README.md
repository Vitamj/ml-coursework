# Seoul Bike Data Analysis

## Tech stack
Python, pandas, NumPy, matplotlib, seaborn, scikit-learn

## Overview
This project explores the Seoul Bike Sharing dataset and focuses on data preprocessing, exploratory data analysis, numerical operations, similarity metrics, smoothing techniques, and basic clustering.

## Goal
Analyze bike rental data, prepare features for downstream tasks, and explore patterns in the dataset through preprocessing, visualization, and simple unsupervised methods.

## What was done
- loaded and inspected a real-world tabular dataset
- processed categorical and date-based features
- explored relationships between variables using correlation and similarity measures
- applied moving average and Gaussian smoothing
- performed basic clustering analysis
- visualized key patterns in the data

## Result
Built a structured workflow for tabular data analysis that combines preprocessing, feature handling, exploratory analysis, and introductory clustering techniques.
