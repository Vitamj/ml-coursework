# Red-Black Tree Film Recommendation

## Overview
This project implements a red-black tree for storing film ratings, updating average scores, and recommending films with ratings closest to a target value.

## Tech stack
C++, data structures, algorithms

## Goal
Build a balanced search tree that supports efficient insertion, lookup, rating updates, and recommendation by average rating.

## What was done
- implemented a red-black tree from scratch
- added insertion with balancing via rotations and recoloring
- supported updating film ratings and recomputing average scores
- implemented search by film title
- implemented in-order traversal for nearest-rating recommendation
- maintained balanced tree properties for efficient operations

## Result
Built a balanced tree-based recommendation structure that combines core red-black tree operations with a simple rating-based recommendation mechanism.
