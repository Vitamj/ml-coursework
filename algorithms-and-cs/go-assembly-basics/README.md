# Go Assembly Basics

## Overview
This project contains low-level implementations of several basic algorithms in Go assembly.

## Tech stack
Go, assembly, low-level programming, algorithms

## Goal
Practice low-level programming by implementing core routines directly in assembly and understanding how data, loops, arithmetic, and slice access work at a lower abstraction level.

## What was done
- implemented lower bound search in assembly
- implemented iterative Fibonacci computation in assembly
- implemented slice summation in assembly
- worked with registers, branching, memory offsets, and function arguments
- handled low-level operations on slices and scalar values
- practiced algorithm implementation in Go assembly syntax

## Result
Built a small collection of low-level algorithm implementations in Go assembly to strengthen understanding of memory layout, control flow, and performance-oriented programming.
