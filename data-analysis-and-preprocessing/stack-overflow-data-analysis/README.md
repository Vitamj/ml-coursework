# Stack Overflow Data Analysis

## Tech stack
Python, pandas, matplotlib, seaborn, BeautifulSoup, NLTK

## Overview
This project explores Stack Overflow question, answer, and tag data with a focus on text cleaning, dataset merging, exploratory analysis, and distribution-based insights.

## Goal
Analyze the structure of Stack Overflow content, identify popular programming topics, inspect rating distributions, and prepare textual data for further downstream analysis.

## What was done
- loaded and merged question, answer, and tag datasets
- cleaned raw HTML text and normalized textual content
- applied stopword removal and lemmatization
- analyzed question score distribution and outliers
- identified the most frequent tags
- compared the distribution of questions across programming languages
- visualized rating, tag, and text-length patterns

## Result
Built a structured analytical workflow for Stack Overflow data, combining preprocessing, exploratory analysis, and topic-level inspection across programming-related content.
