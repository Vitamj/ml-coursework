# Seoul Bike Temporal Analysis

## Tech stack
Python, pandas, NumPy, plotly, matplotlib, scikit-learn

## Overview
This project explores temporal patterns in the Seoul Bike Sharing dataset, with a focus on aggregation, seasonality, anomaly detection, missing value handling, external feature enrichment, and usage segmentation.

## Goal
Analyze how bike rental activity changes over time and identify meaningful patterns related to weekdays, weekends, holidays, weather conditions, and demand intensity.

## What was done
- aggregated bike rental data by day, week, and month
- explored weekday vs weekend rental patterns
- detected anomalies using the IQR method
- merged rental data with holiday information
- handled missing values in weather-related features
- segmented usage intensity across time periods
- analyzed the relationship between weather conditions and bike demand


## Result
Built a structured exploratory workflow for temporal and seasonal analysis of bike rental demand, combining aggregation, anomaly detection, enrichment, and segmentation techniques.
