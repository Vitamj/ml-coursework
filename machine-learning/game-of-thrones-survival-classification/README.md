# Game of Thrones Survival Classification

## Overview
This project focuses on binary classification for predicting whether a character from the Game of Thrones universe survives or dies.

## Tech stack
Python, pandas, NumPy, matplotlib, seaborn, scikit-learn

## Goal
Prepare character-level data, engineer useful features, and build a classification pipeline for survival prediction.

## What was done
- explored the character dataset and target distribution
- handled missing values and categorical features
- created and transformed predictive features
- removed irrelevant columns
- analyzed feature relationships with the target
- prepared data for model training
- built a classification workflow using scikit-learn

## Result
Built a structured classification pipeline on a real-world character dataset, combining preprocessing, exploratory analysis, feature engineering, and supervised learning.
