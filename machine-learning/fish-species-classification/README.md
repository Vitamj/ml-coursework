# Fish Species Classification

## Overview
This project focuses on multiclass fish species classification using numerical features such as weight, length, height, and width.

## Tech stack
Python, pandas, NumPy, scikit-learn, matplotlib

## Goal
Train and compare classification models for predicting fish species from structured tabular data, with proper validation, hyperparameter tuning, and error analysis.

## What was done
- performed initial exploratory analysis of the dataset
- encoded target labels and split the data with stratification
- trained Logistic Regression models with polynomial features
- manually tuned regularization strength using validation MAE
- evaluated SVM models with different kernels and hyperparameters
- compared Logistic Regression and SVM performance on the test set
- analyzed results with classification metrics and a confusion matrix

## Result
Built and compared multiclass classification pipelines for tabular data and explored the effect of model complexity, regularization, and kernel choice on classification quality.
